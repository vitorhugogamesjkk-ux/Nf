Projeto: Consulta NF (React + Vite)
================================
Como usar (local):
1. Instale Node.js (v18+ recomendado).
2. No terminal, rode:
   npm install
   npm run dev
3. Abra http://localhost:5173

Como publicar na Vercel:
1. Crie um repositório no GitHub e envie todos os arquivos.
2. Acesse https://vercel.com, faça login e clique em "New Project".
3. Importe o repositório.
4. Vercel detecta React/Vite automaticamente. Deploy.
